# Nyx - A Aventura do Saber 🎮📚

Este repositório contém o código-fonte e os recursos do videojogo educativo **Nyx - A aventura do saber**, desenvolvido com o motor de jogo **Unity Engine** e a linguagem de programação **C#**.

## 🚀 Como Executar o Jogo (Build Final)
1. Descarregue a pasta compactada `PAP - NYX.rar`.
2. Extraia o conteúdo para uma pasta à sua escolha no computador.
3. Abra a pasta extraída e execute o ficheiro **`Nyx - A aventura do saber.exe`**.

*Nota: O ficheiro executável dependente da pasta `_Data` na sua raiz para conseguir arrancar corretamente.*

## 🛠️ Como Abrir o Código-Fonte (Unity)
1. Instale o **Unity Hub** no computador.
2. Extraia a pasta do código-fonte do projeto.
3. No Unity Hub, clique em **Add** > **Add project from disk** e selecione a pasta raiz que contém as diretorias `Assets` e `ProjectSettings`.
4. Abra o projeto através do Hub.

## ⚙️ Tecnologias
- Motor de Jogo: Unity Engine
- Linguagem: C#
- Persistência: Ficheiros Locais JSON / PlayerPrefs (Sem dependência de base de dados externa)
